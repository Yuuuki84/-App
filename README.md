# お天気アプリ App 🌤️

日本語の都市名（都道府県・市区町村）を入力して、  
**現在の天気情報を取得・表示するシンプルな Web アプリ**です。

HTML / CSS / JavaScript のみで構成されており、  
**GitHub Pages 上でブラウザからそのまま動作確認**できます。

## 📸 アプリ概要

- 日本語入力に対応（例：大阪、和歌山、神戸 など）
- OpenWeather API を利用して現在の天気を取得
- 主要な気象情報をカード形式で表示
- レスポンシブ対応（PC / スマートフォン）
- ダークモード切り替え対応
- 最近検索した都市の履歴を保存（LocalStorage）

---

## ✅ 主な機能

- 🌍 **都市名検索**
  - 日本語の都市名から天気を取得
  - Geocoding API → 天気 API の 2 段階検索

- 🌡️ **詳細な天気情報表示**
  - 天気概要（アイコン / 説明）
  - 気温 / 体感気温 / 最低・最高気温
  - 湿度 / 気圧 / 視程
  - 風速 / 風向
  - 雲量 / 雨量 / 雪量
  - 日の出 / 日の入り
  - 緯度・経度
  - 観測日時

- 📍 **現在地からの天気取得**
  - ブラウザの位置情報 API を利用

- 🔁 **単位切り替え**
  - ℃（metric） / ℉（imperial）

- 🌓 **テーマ切り替え**
  - ライト / ダークモード（設定は保存されます）
